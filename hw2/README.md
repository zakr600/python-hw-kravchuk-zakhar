# Дз2 Кравчук Захар

В папке latexgen лежит описанная в условии библиотека - генератор LaTeX

## Артефакты ДЗ:
### 2.1:
- используя другой модуль (generate_pdf.py), импортирую свою библиотеку получаю на выход файл result.tex
- используя pdflatex, и подавая result.text на вход, получаю итоговый файл: result.pdf

### 2.2:
- добавляю поддержку работы с картинками, обновляю tex и pdf файлы из прошлой задачи
- собираю и публикую свою библиотеку в pypi репозитории: https://pypi.org/project/latexgen-zakr600/0.1.0/#description

### 2.3:
- Реализую Dockerfile для воспроизводимости без лишних бинарных зависимостей
- **Пример использования:**
docker build -t latexgen .
docker run --rm -v $(pwd):/app latexgen
